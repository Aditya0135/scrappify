How It Works

Step 1: Provide the Review Link

Paste the Flipkart product review link into the input field on our homepage.

  • The link is analyzed for validity and structure.
  • Ensuring the correct format helps in accurately scraping data.
  • Pre-validation steps ensure smoother subsequent processing.
  • Incorrect links are flagged for user correction.
Show More

Step 2: Scraping Reviews

Our algorithm, powered by BeautifulSoup, efficiently scrapes reviews.

  • BeautifulSoup parses HTML to locate and extract review content.
  • It navigates through DOM structures using tags and attributes.
  • Data is then cleaned, removing unnecessary HTML tags and whitespace.
  • Edge cases, like missing data or inconsistent layouts, are handled robustly.
Show More

Step 3: Analyzing Sentiments

Reviews are analyzed using VADER sentiment analysis.

  • VADER is a lexicon-based sentiment analysis tool optimized for social media.
  • It computes polarity scores to classify sentiments as positive, neutral, or negative.
  • Complexity is managed by handling negations and degree modifiers.
  • Edge cases, like sarcasm, are minimally addressed but noted for potential refinement.
Show More

Step 4: Presenting Data through Graphs

The analyzed data is visualized through a pie chart, highlighting sentiment distribution.

  • The graph displays the percentage of positive, neutral, and negative reviews.
  • This visualization helps users quickly understand the overall sentiment.
  • It is dynamically generated based on real-time analyzed data.
  • Interactive elements in the graph enhance data interpretation.
